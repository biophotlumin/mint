MINTÔÉÅ

Modified Intraneuronal Nanoparticle Tracking (MINT) is a Python script used to extract intraneuronal transport parameters from microscopy data.

It relies on trackpy to generate trajectories from video files, from which transport parameters are then extracted and statistically tested.

Its purpose is to automate workflow to the point where an input of raw video files results in an output of graphs and p-values.
